# Python Mini Projects 🐍

A collection of beginner-friendly Python projects built while learning Python from the **CodeWithHarry Python Course**. These projects helped me strengthen my understanding of Python fundamentals by applying concepts to real-world problems.

## 📌 Projects Included

### 1. 💧 Water Notification Reminder
A simple desktop reminder application that reminds users to drink water at regular intervals.

**Concepts Used**
- Loops
- Time module
- Notifications
- Functions

**Skills Learned**
- Scheduling tasks
- Working with time delays
- Desktop notifications

---

### 2. ⌨️ Typing Speed Test
A Python application that measures typing speed and accuracy.

**Concepts Used**
- Time module
- String comparison
- User input
- Functions

**Skills Learned**
- Measuring execution time
- Calculating WPM (Words Per Minute)
- Accuracy calculation

---

### 3. 🐍 Snake Water Gun Game
A Python implementation of the classic Snake-Water-Gun game (similar to Rock-Paper-Scissors).

**Concepts Used**
- Random module
- Conditional statements
- Loops
- Functions

**Skills Learned**
- Game logic
- Random number generation
- Score tracking

---

### 4. ❓ Quiz Game
A console-based quiz application with multiple-choice questions and score calculation.

**Concepts Used**
- Dictionaries
- Lists
- Loops
- Conditional statements

**Skills Learned**
- Data structures

- Score management

### 5. 📄 PDF Merger
A utility that merges multiple PDF files into a single PDF document.

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/python-mini-projects.git
```

2. Navigate to the project folder

```bash
cd python-mini-projects
```

3. Install required libraries (if needed)

```bash
pip install PyPDF2 plyer
```

4. Run any project

```bash
python filename.py
```
